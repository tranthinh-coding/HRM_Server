## HR Manager

Quản lý nhân sự, tiền lương nhân viên

### Contributing
#### Require
**PHP8.^** & **pnpm**

Install Laravel packages
```shell
composer install
```

Node packages
```shell
pnpm install
```

Generate Key
```shell
php artisan key:generate
```

Migrate
```shell
php artisan migrate
```

Passport Install
```shell
php artisan passport:install
```

Start server
```shell
php artisan serve

# of php artisan serve --host=localhost --port=80
# to running on http://localhost
```
